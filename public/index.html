API

Overview

The Apache Cordova emulator can be invoked with a web request. The request options are provided as query string parameters and the response will check for all dependencies before opening the Ripple emulator.

Request

  • http://emulate.cordova.io?url=<url>&platform=<platform>

Parameters

URL

  • The url is the address of your local or public web app.
  • The url is a required parameter.

  • Examples:

  • http://emulate.cordova.io?url=cordova.io
  • http://emulate.cordova.io?url=http://cordova.io
  • http://emulate.cordova.io?url=http://cordova.io/images

Platform

  • The platform is the environment to emulate.
  • The platform is an optional parameter.

  • Emulates latest version:

  • http://emulate.cordova.io?url=cordova.io&platform=cordova
  • http://emulate.cordova.io?url=cordova.io&platform=phonegap

  • Emulates specific version:

  • http://emulate.cordova.io?url=cordova.io&platform=cordova-1.6.0
  • http://emulate.cordova.io?url=cordova.io&platform=phonegap-1.0.0
